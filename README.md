# CSE340 W04 Scaffold

This is a minimal scaffold to implement the W04 assignment features (create/edit organizations, projects, categories; assign categories to projects).

Run:

```powershell
npm install
npm start
```

Open http://localhost:3000
# CSE340 Assignment App

Quick start:

1. Install dependencies

```bash
npm install
```

2. Initialize the SQLite DB

```bash
mkdir data
sqlite3 data/database.sqlite < db/init.sql
```

3. Start the app

```bash
npm start
```

Open http://localhost:3000/categories
